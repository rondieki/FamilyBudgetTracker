# Family Budget Tracker

A modern full-stack budgeting application built using React, Django REST Framework, and PostgreSQL.

## Features

- User Authentication
- Budget Management
- Expense Tracking
- Savings Goals
- Financial Reports
- Notifications
- AI-powered Budget Insights (Coming Soon)

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- Django
- Django REST Framework

### Database
- PostgreSQL

### Deployment
- Vercel
- Render
